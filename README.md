# manipulacao-de-dom

Projeto desenvolvido com foco em prática de manipulação do DOM utilizando JavaScript puro.

A aplicação permite criar elementos HTML dinamicamente na tela, selecionar elementos já criados e adicionar novos elementos dentro deles, simulando uma pequena estrutura visual baseada em HTML.

Além da criação dos elementos, também é possível:
- alterar textos
- adicionar classes do TailwindCSS
- resetar estilos
- remover elementos
- visualizar informações do elemento selecionado

O projeto foi criado como forma de praticar:
- manipulação de elementos no DOM
- eventos de clique
- criação dinâmica de componentes
- seleção de elementos
- alteração de atributos e classes
- organização de lógica com JavaScript

---

## 🚀 Funcionalidades

- Criação dinâmica de tags HTML
- Estrutura de elementos filhos
- Seleção de elementos via clique
- Alteração de texto
- Adição de estilos com TailwindCSS
- Reset de estilos e textos
- Remoção de elementos
- Painel com informações do elemento selecionado
